# Physical AI & Humanoid Robotics Textbook Documentation

This directory contains the complete content for the "Physical AI & Humanoid Robotics" textbook. The content is organized into 7 chapters plus a comprehensive capstone project that integrates all concepts.

## Chapter Structure

Each chapter directory contains:

- `*.md` files: The main content for each chapter section
- `_category_.json`: Navigation and metadata for Docusaurus

## Chapters

1. **Chapter 1: Introduction to Physical AI and Embodied Intelligence**
   - `embodied-intelligence.md`: Core concepts of embodied intelligence
   - `ai-in-physical-world.md`: Differences between virtual and physical AI

2. **Chapter 2: ROS 2 fundamentals for humanoid robots**
   - `ros2-basics.md`: Core ROS 2 concepts for robotics
   - `humanoid-robot-control.md`: Specific control concepts for humanoid robots

3. **Chapter 3: Robot simulation using Gazebo and Unity**
   - `gazebo-simulation.md`: Gazebo simulation techniques
   - `unity-integration.md`: Unity integration for robotics

4. **Chapter 4: NVIDIA Isaac Sim and Isaac ROS**
   - `isaac-sim-overview.md`: Overview of Isaac Sim capabilities
   - `isaac-ros-integration.md`: Isaac ROS integration

5. **Chapter 5: Vision-Language-Action systems**
   - `computer-vision.md`: Computer vision for robotics
   - `action-systems.md`: Action systems and planning

6. **Chapter 6: Conversational humanoid robots**
   - `dialogue-systems.md`: Dialogue systems for robots
   - `human-robot-interaction.md`: Human-robot interaction principles

7. **Chapter 7: Capstone - Autonomous simulated humanoid robot**
   - `autonomous-humanoid-robot.md`: Complete integration project
